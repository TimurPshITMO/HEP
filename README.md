# HEP: Hypergraph-Evolved Pipelines

**Hypergraph-Evolved Pipelines (HEP)** — это профессиональный фреймворк для автоматического конструирования признаков (**AutoFE**), использующий генетические алгоритмы на основе структуры **функциональных гиперграфов**.

Проект позволяет выявлять скрытые нелинейные зависимости высокого порядка в данных и синтезировать новые информативные признаки, которые в дальнейшем могут использоваться для качественного скачка в точности решающей модели.

---

## 🧬 Почему HEP?

В отличие от классического генетического программирования (GP) на основе деревьев, гиперграфовая модель обладает **высокой представительской плотностью**:

1.  **Борьба с Code Bloat**: k-арные взаимодействия (связи 3, 4 и более признаков) описываются единым логическим блоком — гиперребром, а не глубоким деревом вложенных функций.
2.  **Прозрачность**: Каждое выученное гиперребро — это понятная математическая формула, доступная для экспертной проверки.
3.  **Вычислительная мощь**: Встроенная система сигнатур (MD5 хэширование топологии) гарантирует, что ни одна идентичная структура не будет вычислена дважды.

## 🛠 Установка

Проект использует `Poetry` для управления зависимостями.

```bash
# Клонирование репозитория
git clone https://github.com/YourRepo/HEP.git
cd HEP

# Установка через Poetry (рекомендуется)
poetry install

# Или через обычный pip
pip install -r requirements.txt
```

## 🚀 Быстрый старт

Самый наглядный способ познакомиться с HEP — интерактивный блокнот: 
👉 **[notebooks/hep_showcase.ipynb](notebooks/hep_showcase.ipynb)** *(откройте через VS Code или Jupyter)*.

### Пример использования API:
```python
from hep_engine import FitnessEvaluator, EvolutionaryOptimizer

# 1. Инициализация оценки (X, y - ваши данные)
evaluator = FitnessEvaluator(X_train, y_train, complexity_penalty=0.01)

# 2. Запуск поиска признаков
optimizer = EvolutionaryOptimizer(pop_size=20)
final_pop = optimizer.run(evaluator, n_generations=15)

# 3. Получение лучшего гиперграфа
best_hg = final_pop.best().genome
X_new = best_hg.transform(X_test) # Синтез признаков для теста
```

## 📂 Карта проекта

| Модуль | Описание |
| :--- | :--- |
| `hep_engine/core.py` | Математическое ядро: Гиперребра, MD5-сигнатуры и NumPy трансформации. |
| `hep_engine/evolution.py` | Генетическая логика: особи, популяции и биологические операторы. |
| `hep_engine/optimizer.py` | Основной цикл ГА с поддержкой таймаутов и элитизма. |
| `hep_engine/visualizer.py`| Отрисовка гиперграфов "пузырьками" (Euler diagrams) через `HyperNetX`. |
| `tests/` | Полный набор Unit и интеграционных тестов. |
