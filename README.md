# HEP: Hypergraph-Evolved Pipelines

**Hypergraph-Evolved Pipelines** — это инновационный фреймворк для автоматического конструирования признаков (AutoFE), использующий генетические алгоритмы на основе динамических гиперграфов.

Этот проект позволяет обнаруживать сложные нелинейные зависимости в данных и синтезировать новые информативные признаки, повышающие качество моделей машинного обучения.

## 🚀 Основные возможности

- **Эволюция гиперграфов:** Автоматический поиск оптимальных математических трансформаций признаков.
- **Интерпретируемость:** Полученные признаки легко декодируются в понятные математические формулы.
- **Интеграция со sklearn:** Легкое встраивание в существующие пайплайны машинного обучения.
- **Визуализация:** Наглядное представление структуры гиперграфов и процесса их эволюции.

## 📂 Структура проекта

- `hep_engine/` — Ядро фреймворка (эволюционный процесс, гиперграфы, вычисление фитнеса).
- `notebooks/` — Демонстрационные материалы.
    - `hep_showcase.ipynb` — **Основной showcase ноутбук** с подробными пояснениями.
- `tests/` — Юнит-тесты для проверки корректности работы ядра.

## 🛠 Установка и запуск

1. Убедитесь, что у вас установлен Python 3.8+.
2. Установите необходимые зависимости:
   ```bash
   pip install numpy pandas matplotlib scikit-learn networkx
   ```
3. Для визуализации гиперграфов рекомендуется установить:
   ```bash
   pip install hypernetx
   ```
